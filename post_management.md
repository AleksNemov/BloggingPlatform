# Создание платформы для ведения блога

## Функции которыми должна обладать платформа

1. Личный кабинет

2. Возможность создания различных типо контента:

    - Текстовый контент
    - Видеоконтент
    - Аудиоконтент
    - Графический контент

3. Интеграция с социальными сетями

4. Шаблоны оформления
